# 资料列表

## docs

Docs目录下包含课程的主要文档资源，对应[网页](https://tugraph-contrib.github.io/tugraph-course)上展示的内容

## labs

Labs文件夹下包含课程中的实验资源，具体目录结构如下：
- `minigu/` 包含 MiniGU 图数据库开发实战系列的实验资源
- `tugraph/` 包含 TuGraph-DB 图查询与图算法实验

## misc

misc文件下包含一些其他资源：
- `autograding/` 包括自动评分脚本和自动代码查重脚本
- `zips/` 包括社区贡献的作业设计和自动评分脚本的压缩包，如原用于上海交通大学 AI3602 数据挖掘课程的编程作业
